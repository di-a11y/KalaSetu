# KalaSetu 🎨  
Connecting Indian artisans with buyers through a digital marketplace.

## 🚀 Setup
1. Clone this repo  
2. Go inside the `frontend` folder  
3. Install dependencies: `npm install`  
4. Run the project: `npm run dev`

## 📁 Folder Structure
- `frontend/` — React + Vite app  
- Backend will be added later
